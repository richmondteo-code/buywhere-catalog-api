MCP Integration

Connect BuyWhere directly to Claude, Cursor, and any MCP-compatible AI agent.

What is MCP?

The Model Context Protocol (MCP) enables AI agents to connect to external data sources and tools. BuyWhere provides first-class MCP support so your AI agents can search and compare products natively.

Setup

Add BuyWhere to your MCP configuration:

{
  "mcpServers": {
    "buywhere": {
      "command": "npx",
      "args": ["-y", "@buywhere/mcp-server"],
      "env": {
        "BW_API_KEY": "bw_live_your_api_key_here"
      }
    }
  }
}

Available Tools

Tool Description
search_products Search for products by query across all US retailers
get_product Get detailed product information by ID
compare_prices Compare prices for a specific product across retailers
get_deals Get current deals and promotions

Example Usage

Once configured, your agent can search products naturally:

User: Find me the best price on Sony WH-1000XM5 headphones

Agent calls: search_products({
  query: "Sony WH-1000XM5 headphones",
  country: "US"
})

Response: Found 3 results starting at $279.99 from Amazon
