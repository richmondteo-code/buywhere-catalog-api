Quickstart

Get up and running with the BuyWhere API in under 5 minutes.

1. Get Your API Key

Sign up for a free account at buywhere.io/register to get your API key.

bw_live_your_api_key_here

2. Make Your First Request

Search for products using a simple cURL request:

curl -s "https://api.buywhere.io/v1/products/search?q=wireless+headphones&country=US&limit=3" \
  -H "X-API-Key: bw_live_your_api_key_here"

3. Parse the Response

You'll receive a JSON response with product details:

{
  "object": "list",
  "query": "wireless headphones",
  "total_estimated": 142,
  "results": [
    {
      "product_id": "prod_amazon_sony_wh1000xm5_us",
      "title": "Sony WH-1000XM5 Wireless Noise Cancelling Headphones",
      "price": { "amount": 449.00, "currency": "USD" },
      "merchant": { "name": "Sony Official Store", "platform": "amazon" },
      "source_url": "https://amazon.com/dp/B0CHX1W7V2"
    }
  ]
}
