Authentication

All BuyWhere API requests require authentication via an API key.

Getting Your API Key

Sign up for a free account at buywhere.io/register to get your API key. Your API key will be available in your dashboard.

Using Your API Key

Include your API key in the X-API-Key header of every request:

curl -s "https://api.buywhere.io/v1/products/search?q=headphones" \
  -H "X-API-Key: bw_live_your_api_key_here"

API Key Types

Type Prefix Usage
Live bw_live_ Production requests
Test bw_test_ Development and testing

Security

  • Keep your API key secure and never commit it to version control
  • Use environment variables to store your API key
  • Rotate your API key if you suspect it has been compromised
  • Use test keys for development to avoid affecting production data
