Code Examples

Ready-to-run code examples in popular languages and frameworks.

cURL

curl -s "https://api.buywhere.io/v1/products/search?q=sony+headphones&country=US&limit=5" \
  -H "X-API-Key: bw_live_your_key"

JavaScript / Node.js

const response = await fetch(
  'https://api.buywhere.io/v1/products/search?q=sony+headphones&country=US',
  {
    headers: {
      'X-API-Key': 'bw_live_your_key'
    }
  }
);
const data = await response.json();
console.log(data.results);

Python

import requests

response = requests.get(
    'https://api.buywhere.io/v1/products/search',
    params={'q': 'sony headphones', 'country': 'US'},
    headers={'X-API-Key': 'bw_live_your_key'}
)
data = response.json()
print(data['results'])

Python with LangChain

from buywhere_langchain import BuyWhereTool
from langchain.agents import initialize_agent, Tool

buywhere = BuyWhereTool(api_key="bw_live_your_key")

tools = [Tool(
    name="search",
    func=buywhere.search,
    description="Search for products by query"
)]

agent = initialize_agent(tools, llm, agent="zero-shot-react-description")
result = agent.run("Find the best price for wireless headphones")

Python with OpenAI Functions

import json
import openai
import requests

def search_products(query: str, country: str = "US") -> str:
    response = requests.get(
        'https://api.buywhere.io/v1/products/search',
        params={'q': query, 'country': country},
        headers={'X-API-Key': 'bw_live_your_key'}
    )
    return response.json()

tools = [{
    "type": "function",
    "function": {
        "name": "search_products",
        "description": "Search for products across retailers",
        "parameters": {
            "type": "object",
            "properties": {
                "query": {"type": "string"},
                "country": {"type": "string", "default": "US"}
            }
        }
    }
}]

messages = [{"role": "user", "content": "Find me wireless headphones"}]
response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=messages,
    tools=tools,
    tool_choice="auto"
)
print(response)
