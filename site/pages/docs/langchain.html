LangChain Integration

Use BuyWhere as a tool in your LangChain agent pipelines.

Installation

pip install buywhere-langchain

Quick Start

from langchain.agents import initialize_agent, Tool
from langchain.llms import OpenAI
from buywhere_langchain import BuyWhereTool

# Initialize the BuyWhere tool
buywhere_tool = BuyWhereTool(api_key="bw_live_your_key")

# Create tools list
tools = [
    Tool(
        name="product_search",
        func=buywhere_tool.search,
        description="Search for products by query"
    ),
    Tool(
        name="compare_prices",
        func=buywhere_tool.compare,
        description="Compare prices for a specific product"
    )
]

# Initialize agent
llm = OpenAI(temperature=0)
agent = initialize_agent(tools, llm, agent="zero-shot-react-description")

# Use the agent
result = agent.run("Find the best price for Sony WH-1000XM5 headphones")

BuyWhereTool Methods

Method Description
search(query, country='US', limit=10) Search for products
compare(product_id) Compare prices across retailers
get_deals(category=None) Get current deals

Example: Product Research Agent

from langchain.agents import initialize_agent, Tool
from langchain.llms import OpenAI
from buywhere_langchain import BuyWhereTool

buywhere = BuyWhereTool(api_key="bw_live_your_key")

prompt = """You are a product research assistant.
Use the search tool to find products matching the user's criteria.
Present the results with prices and retailer information.

When comparing products:
- List key specifications
- Compare prices across retailers
- Note any deals or promotions"""

agent = initialize_agent(
    buywhere.as_tools(),
    OpenAI(temperature=0),
    agent="zero-shot-react-description",
    verbose=True
)
