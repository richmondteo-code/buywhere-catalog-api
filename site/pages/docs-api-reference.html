GET /v1/products/search

Search for products using natural language queries. Returns a paginated list of matching products sorted by relevance.

Query Parameters

Parameter Type Description
q string Search query (required)
country string Country code: US, SG, MY, ID, VN, PH (default: US)
category string Category slug filter
min_price number Minimum price in local currency
max_price number Maximum price in local currency
platform string Filter by platform: amazon, walmart, target, bestbuy
limit integer Results per page, 1-50 (default: 10)
cursor string Pagination cursor from the previous next_cursor value
page integer Backward-compatible fallback for older page-based clients. Prefer cursor for new integrations.

Example Request

curl -s "https://api.buywhere.io/v1/products/search?q=wireless+headphones&country=US&limit=3" \
  -H "X-API-Key: bw_live_xxx"
