How It Works

BuyWhere gives AI agents a unified API to search, compare, and retrieve product data across Singapore's top e-commerce platforms.

1

Query the API

Your AI agent sends a natural language or structured query to BuyWhere's REST API. Specify the country code (SG, MY, TH, ID, PH) to target specific SEA markets.

curl -s "https://api.buywhere.io/v1/products/search?q=wireless+headphones&country=SG" \
  -H "X-API-Key: bw_live_xxx"
2

Get Normalized Results

BuyWhere searches across Shopee, Lazada, FairPrice, Carousell, Qoo10, and other platforms — and returns a consistent JSON schema regardless of the source.

3

Agent Takes Action

Your agent can compare prices, check availability, or redirect users to the purchase URL — all without scraping or site-specific logic.

Full Example

import requests

url = "https://api.buywhere.io/v1/products/search"
headers = {"X-API-Key": "bw_live_xxx"}
params = {
    "q": "samsung galaxy s25",
    "country": "SG",
    "limit": 10
}

response = requests.get(url, headers=headers, params=params)
products = response.json()["products"]

for p in products:
    print(f"{p['name']} — {p['price']} at {p['source']}")
