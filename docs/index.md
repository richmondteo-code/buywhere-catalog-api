# BuyWhere API Documentation

Welcome to the BuyWhere API — the agent-native product catalog for AI commerce across Singapore and Southeast Asia.

## What is BuyWhere?

BuyWhere provides a unified API to search, compare, and purchase products across multiple e-commerce platforms including Shopee, Lazada, Carousell, Amazon, and more — all indexed and queryable in real-time.

```
https://api.buywhere.ai
```

## Core Features

| Feature | Description |
|---------|-------------|
| **Search** | Full-text product search across 1M+ products |
| **Compare** | Cross-platform price comparison |
| **Best Price** | Find the cheapest listing for any product |
| **Categories** | Browse by category with product counts |
| **Deals** | Current discounts and price drops |
| **Export** | Bulk data export in CSV or JSON |

## Supported Platforms

Shopee SG, Lazada SG, Carousell, Amazon SG, Fairprice, Giant, Cold Storage, Challenger, IKEA, Courts, Harvey Norman, Decathlon, Uniqlo, Nike, Sephora, Watson's, and more.
