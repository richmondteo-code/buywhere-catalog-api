Roadmap

What we are building next for the BuyWhere product catalog API. Updated regularly.

🟢 In Progress

  • Malaysia coverage expansion In Progress
    Adding Shopee MY and Lazada MY product coverage. Expected Q2 2026.
  • MCP server reference implementation In Progress
    Model Context Protocol server for BuyWhere. Lets agent frameworks like Claude directly use BuyWhere as a native tool.
  • Deal alerts webhook API In Progress
    Subscribe to price drop alerts on specific products or categories. Push to your agent via webhook.

🔵 Exploring

  • Vietnam coverage Exploring
    ShopeeVN and LazadaVN product indexing. Depends on Malaysia launch success.
  • Affiliate commission dashboard Exploring
    Merchant and developer portal showing affiliate earnings, conversion rates, and commission history.
  • Structured product attributes Exploring
    Normalized product specifications (screen size, RAM, flavor, etc.) enabling precise filtering and comparison beyond price.

🟢 Shipped

  • Singapore product catalog Shipped
    Shopee SG, Lazada SG, Amazon.sg, and 40+ Singapore merchant products indexed and normalized.
  • Python SDK v1 Shipped
    Python SDK with full API coverage. Install via pip: pip install buywhere
  • Private developer beta Shipped
    Developer beta program open for AI agent developers. Apply via hello@buywhere.ai.
  • Price comparison endpoint Shipped
    Compare prices across platforms for the same product by product ID.

API Stability Commitment

BuyWhere does not make breaking changes to the API without 90 days notice. We increment versions for breaking changes and maintain backwards compatibility within major versions. Backwards-compatible additions (new fields, new endpoints) are deployed continuously without notice.

Questions about the roadmap? Email hello@buywhere.ai.
