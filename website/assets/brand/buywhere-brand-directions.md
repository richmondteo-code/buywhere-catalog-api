# BuyWhere Brand Directions

Status: Draft for board review on [BUY-571](/BUY/issues/BUY-571)

## Brand intent

BuyWhere should read as infrastructure, not retail. The brand needs to signal:

- trusted catalog infrastructure for AI agents
- developer clarity and API precision
- regional credibility in Singapore with global ambition
- neutrality across merchants, agents, and marketplaces

## Recommended direction

Recommend **Concept 01: Signal Grid**.

Why:

- strongest infrastructure feel
- easiest to scale into favicon, API docs, deck cover, and wordmark lockups
- visually distinct from generic shopping or fintech marks

## Concept 01: Signal Grid

Files:

- `concept-01-signal-grid.svg`

Idea:

An abstract `B` built from two routed lanes inside a rounded grid tile. It suggests indexing, routing, and machine-readable structure.

Palette:

- Ink: `#0F172A`
- Grid Blue: `#2563EB`
- Mint Signal: `#14B8A6`
- Cloud: `#E2E8F0`

Typography:

- Headlines: `Space Grotesk`
- UI / body: `Inter`
- Code: `JetBrains Mono`

## Core guidance

- Avoid cart, bag, tag, and barcode tropes.
- Avoid consumer-commerce red and discount-led visual language.
- Keep the mark geometric and compact enough for affiliate network application forms.
- Use calm, technical color contrast over loud startup gradients.
