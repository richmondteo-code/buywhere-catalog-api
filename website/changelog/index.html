Product Updates

Changelog

API updates, new endpoints, and improvements. We announce breaking changes 90 days in advance.

  • April 27, 2026
    NewProduct search with semantic relevance scoring

    Added relevance_score field to all search results. Scores range 0-100 and reflect semantic match quality against the query intent.

    • New field: relevance_score (integer, 0-100)
    • Default behavior unchanged — results still ranked by composite score
    • Use sort=relevance to sort purely by semantic relevance
  • April 20, 2026
    ImprovedHourly freshness guarantee for Singapore merchants

    All Singapore merchant product data now refreshes hourly. Previously, refresh intervals varied by merchant tier.

    • Price, stock, and metadata updates now within 60 minutes of source change
    • New last_fetched_at field on all product responses
  • April 14, 2026
    NewPrice comparison endpoint

    New POST /v1/products/compare endpoint accepts product IDs and returns a structured price comparison across platforms.

    • Returns cheapest, most expensive, average price
    • Includes per-platform affiliate links
    • Available in private beta — contact hello@buywhere.ai for access
  • April 8, 2026
    BreakingResponse shape v2 migration complete

    Legacy v1 response shape removed. All endpoints return v2 shape as documented. Migration guide available in docs.

    • Top-level results array replaced with data
    • Pagination cursor renamed from next_page to next_cursor
    • v1 endpoints decommissioned — requests to v1 routes now return 410 Gone
