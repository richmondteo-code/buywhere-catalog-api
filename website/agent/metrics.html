Agent Response Time Budget

For a smooth user experience, allocate response time as follows:

Total agent response target: <2 seconds
├── User query parsing: ~100ms
├── BuyWhere API call: ~150ms (p50)
├── Result processing: ~50ms
└── Response formatting: ~100ms
Remaining buffer: ~1.6s

Quality Indicators in Response

Field Agent Usage
buywhere_score Primary quality ranking (0-1). Score >= 0.8 = strong recommendation.
confidence Query match certainty (0-1). Low confidence warrants disclaimer.
merchant.reliability_score Merchant trustworthiness (0-1). Score < 0.7 = flag concern.
availability Stock status. Never recommend out_of_stock items.
metadata.freshness Data timestamp. Older = mention potential price changes.
