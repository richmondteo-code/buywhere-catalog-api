# BuyWhere — Product Price Comparison API and MCP Server

## Overview

BuyWhere is an AI-native product price comparison API and MCP server that enables developers to build shopping agents, price tracking tools, and deal discovery applications. Verified product pricing data from 500+ retailers across US and Southeast Asia.

## Key Features

### Multi-Platform Coverage
- **US Retailers**: Amazon, Best Buy, Walmart, Target, Costco, Newegg, B&H Photo
- **SEA Retailers**: Shopee, Lazada (Singapore, Malaysia, Thailand, Vietnam, Philippines, Indonesia)
- **500+ retailers** in USD, SGD, MYR, THB, VND, PHP, IDR

### MCP Server for AI Agents
BuyWhere provides an MCP (Model Context Protocol) server — AI agents connect natively without custom code.

**Available MCP Tools:**
- `search_products` — Full-text product search with filters
- `get_product` — Product details by ID
- `compare_products` — Cross-merchant comparison
- `get_deals` — Discounted products
- `find_best_price` — Cheapest listing finder
- `list_categories` — Category browsing

### REST API
Full REST API for direct integration — product search, real-time price comparison, price history, and merchant lookup.
