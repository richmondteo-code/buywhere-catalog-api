# BuyWhere — Product Price Comparison API

## Overview

BuyWhere is an AI-native product price comparison API and MCP server that enables developers to build shopping agents, price tracking tools, and deal discovery applications.

## Features

### Multi-Platform Coverage
- **US**: Amazon, Best Buy, Walmart, Target, Costco, Newegg, B&H Photo
- **SEA**: Shopee, Lazada (SG, MY, TH, VN, PH, ID)
- **500+ retailers** in USD, SGD, MYR, THB, VND, PHP, IDR

### MCP Server for AI Agents
AI-native MCP tools for Claude, Cursor, and other MCP clients.

**Tools:** search_products, get_product, compare_products, get_deals, find_best_price, list_categories

### REST API
Product search, real-time price comparison, price history, merchant lookup.
